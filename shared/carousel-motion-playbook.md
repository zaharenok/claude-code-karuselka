# Carusel — Motion Playbook (русский)

Руководство для **carusel-motion-director**: как анализировать slide-01 и решать, как его оживить.

## Обязательный шаг: посмотреть картинку

Агент **обязан** открыть и проанализировать:

`carusel-memory/output/slides/slide-01.png`

Без визуального разбора нельзя писать промпт.

## Что зафиксировать в анализе

1. **Содержимое** — человек / продукт / UI / абстракция / коллаж / текст на макете
2. **Настроение** — энергичный hook / спокойный / премиум / игривый
3. **Точки движения** — что можно двигать, не ломая композицию
4. **Запреты** — что нельзя менять (лица, логотип, текст, пропорции)
5. **Связь с каруселью** — slide-01 останавливает скролл; движение усиливает hook

## Матрица решений

| На слайде | Стратегия движения | Речь / звук в промпте |
|-----------|-------------------|------------------------|
| **Человек (лицо крупно)** | Микромимика, дыхание, лёгкий наклон головы, моргание | Опционально: короткая фраза 1–3 слова **только если** усиливает hook и уместна теме |
| **Человек (в сцене)** | Parallax фона, лёгкий wind/hair, жест рук | Обычно **без речи** — ambient и motion |
| **Продукт / объект** | Вращение, отблеск, тень, «hero glow» | Без речи; лёгкий whoosh/s shimmer в описании атмосферы |
| **UI / скрин** | Пульс курсора, мягкий scroll glow, highlight | Без речи |
| **Абстракция / градиент** | Течение градиента, частицы, волна света | Ambient pulse; **музыкальный ритм** визуально (пульс 120 BPM) |
| **Коллаж / стикеры** | Парение стикеров 2–3 px, paper wiggle | Игривый loop, без речи |
| **Текст крупно на макете** | **Не анимировать буквы** — только фон и декор вокруг | Без речи |

## Звук и «музыка»

Grok генерирует **видео**; отдельной дорожки может не быть. В промпте описывай:

- **Ритм движения** под будущий звук (пульс, beat-sync light)
- **Атмосферу** — «лёгкий cinematic ambient», «мягкий electronic pulse»
- Не обещай конкретный трек; опиши **ощущение** soundtrack

Речь добавляй **только** если в анализе `speech_recommended: true` и есть причина (эксперт, эмоциональный hook).

## Loop (зацикливание)

Всегда:

- 5 секунд
- первый кадр = последний кадр
- без hard cuts
- без смены сцены
- движение «залипательное» (hypnotic hammered loop)

## Качество

- Сохранить identity lock: «референс-кадр = закон»
- Не превращать в другой стиль
- Не добавлять новые объекты
- Движение **тонкое**, не цирк
